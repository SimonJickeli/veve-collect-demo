# VeVe Collect — Community App

A free community companion for the VeVe / Collect ecosystem. **Three tools that run live on your real on-chain collection** — paste a Collect Chain wallet address.

## Tools

| Page | What it does | Data |
|------|--------------|------|
| `index.html` | Landing page with live catalog stats | live |
| `sets.html` | **Set Tracker** — owned vs. missing per set (diffed against the full catalog) + completion rewards + **portfolio value & cost-to-complete at live StackR floor** | **live** holdings + **live** StackR floor |
| `mcp.html` | **MCP Optimizer** — your live daily Master Collector Points (VeVe's real formula) + cheapest points to add across the whole catalog, **set-aware**, ranked by **live StackR on-chain floor** | **live** holdings + **live** StackR floor (drop-price fallback) |
| `mint-checker.html` | **Mint Checker** — scan a wallet for special mints (#1, lowest-public, palindromes, repdigits, milestones, 🏆 grail years) | **live** holdings |

## Engine / data files

| File | What it is |
|------|-----------|
| `assets/mcp.js` | Canonical **MCP point formula** (rarity, low-mint = lowest 5% of *public* editions, duplicate diminishing, set completion). Verified vs VeVe's official pages. |
| `assets/mints.js` | **Significant-mints** engine — 7 selectable categories + universe/character grail years. |
| `assets/significant-years.js` | Character + universe milestone years (e.g. Star Wars #1977). |
| `assets/collectscan.js` | **Live client** for the public collectscan (Blockscout) API — fetch a wallet's VeVe NFTs (CORS open, no proxy). |
| `data/catalog.js` | Catalog + sets **exported from the wiki** (`/tmp/export_catalog.sh`) — 1,091 collectibles, 345 sets. |
| `data/floor.js` | **Live StackR on-chain floor snapshot** (`window.FLOOR`) generated by `/tmp/gen_floor.py` — reads the `StackrListings` contract on Base via `base.blockscout.com`, reconciles active listings, converts OMI→USD. Re-run to refresh. |
| `assets/data.js` | Small mock catalog/holdings for the "Use sample" demos. |
| `assets/style.css` | Self-contained dark theme (no CDN). |

## Run it

```bash
# no-cache dev server (so edits always reflect)
python3 app/devserver.py    # serves app/ at http://127.0.0.1:4178
```
(or use the Claude preview: server name `veve-app` in `.claude/launch.json`).

## Live vs. proxied

- ✅ **Live on-chain:** your holdings, mint numbers, rarities, editions (via collectscan, read-only, no login/keys).
- ✅ **Live on-chain:** **floor prices** — the StackR secondary market (`StackrListings` on Base) via `base.blockscout.com`; OMI→USD. Items not currently listed fall back to original drop price (labeled `~` vs `⛓` in the optimizer).
- ✅ **Real:** the MCP point formula and the significant-mints logic.

## Path to production
