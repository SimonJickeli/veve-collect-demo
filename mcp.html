Where do I find my wallet address?
  1. It's your VeVe Collect Chain (OMI) address — starts 0x…, 42 characters. VeVe manages it for you (self-custody isn't live yet), so you only need to read it.
  2. Where it is (if your app has it): in the VeVe app (or veve.me/en/wallet, log in) open your OMI wallet → Receive — the 0x… with the QR code is your address; tap to copy.
  3. Only see “Send OMI”, no address? VeVe is bridging OMI to StackR and is hiding the Receive screen on newer app versions, so the raw 0x… isn't shown in-app there. It's the same address that holds your collectibles and it never changes — once you have it, paste it here and we remember it. (Username auto-connect is coming with the StackR integration.)
  4. Paste it above — we remember it on this device, so you only enter it once. Coming: connect StackR to auto-fill.

How it works. MCP/day uses VeVe's published formula (rarity, low-mint, duplicate diminishing, set completion). 💎/point = cost ÷ points added — lower is cheaper, priced at the cheaper of the StackR ⛓ traded floor or a sanitized VeVe 💠 ask. Blind-box items are priced from a real secondary-market floor only — never the sealed-box price. Not financial advice.
